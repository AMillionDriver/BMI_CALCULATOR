# Kalkulator BMI (Android Jetpack Compose)

Aplikasi sederhana untuk menghitung Body Mass Index (BMI) berdasarkan input tinggi (dalam meter) dan berat badan (dalam kilogram). Project ini dibuat untuk mempelajari dasar-dasar **Jetpack Compose** dan arsitektur folder di Android.

## ✨ Fitur
- Menghitung nilai BMI secara real-time.
- Memberikan kategori status berat badan (Kurus, Normal, Ideal, atau Obesitas).
- Tampilan responsif menggunakan Material Design 3.
- Struktur kode yang rapi (Pemisahan Logika dan UI).

## 🛠️ Tech Stack
- **Kotlin** - Bahasa pemrograman utama.
- **Jetpack Compose** - Toolkit modern untuk membangun UI secara deklaratif.
- **Material 3** - Desain sistem terbaru dari Google.

## 📁 Struktur Project
Aplikasi ini menerapkan pemisahan kode agar lebih mudah dikelola:
- `ui.screen`: Tempat menyimpan UI / Composable (contoh: `BmiScreen.kt`).
- `logic`: Tempat menyimpan fungsi perhitungan dan logika bisnis (contoh: `BmiLogic.kt`).
- `ui.theme`: Pengaturan warna, tipografi, dan tema aplikasi.
